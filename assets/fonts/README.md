# Fonts Directory

Add your Poppins font files here:

**Required Poppins font files:**
- `Poppins/Poppins-Regular.ttf`
- `Poppins/Poppins-Light.ttf`
- `Poppins/Poppins-Medium.ttf`
- `Poppins/Poppins-Bold.ttf`

You can download Poppins fonts from: https://fonts.google.com/specimen/Poppins

To use the fonts:
1. Download the font files and place them in the Poppins/ subdirectory
2. Uncomment the font loading lines in `constants/constants.ts`
3. The fonts will be available throughout the app
